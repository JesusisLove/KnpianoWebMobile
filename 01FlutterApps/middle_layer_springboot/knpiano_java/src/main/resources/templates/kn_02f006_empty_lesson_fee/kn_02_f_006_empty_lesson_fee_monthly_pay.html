空课学费按月支付（使用对象：按月支付课费的学生）

查询达到43节课的学生并生成剩余月份的临时课程

业务说明
  • 按月交费的学生，年度课程目标是43节课
  • 如果学生提前完成了43节课（例如10月份），则11月和12月仍需支付课费
  • 此功能用于查询这些学生，并可生成"虚拟课程"用于课费收取
  • 达成月份：表示学生在第几个月累计达到43节课（例如：12表示在12月达到43节）
  • 在12月之前达到43节课的学生（达成月份 < 12），需要为后续月份生成空月课费
  • 空月课费：提前完成43节课后，剩余月份虽然不上课，但仍需支付剩余月份的每月课费。43节课的总学费 = 每月学费 × 12个月，无论是在6个月、10个月还是12个月上完43节课，都需要支付12个月的学费
